Comandos de la terminal de Git Bash

clear limpia la terminal.
ls -a muestra los archivos y carpetas.
cd nombreCarpeta entra a la carpeta llamada nombreCarpeta.
cd .. sale de la carpeta en la que estamos.
pwd muestra la ruta absoluta de la carpeta en la que estamos.
mkdir nombreCarpeta crea una carpeta llamada nombreCarpeta.
git init inicializa un repositorio de GIT en la carpeta en la que estamos.
code . abre Visual Code Studio en la carpeta en la que estamos.
