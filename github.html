GITHUB

git remote add origin url vincula el repositorio de GIT con el de GITHUB.
git push -u origin rama envía el último commit de la rama a GITHUB y la establece como predeterminada.
git push envía el último commit de la rama en la que estamos a GITHUB.
git pull trae el último commit de la rama en la que estamos de GITHUB.
git branch -u origin/master soluciona el error que puede aparecer al hacer un pull.
git push origin --delete rama borra una rama en GITHUB.
git fetch trae el historial de commits de GITHUB.
git log --online --all muestra el historial de commits de GITHUB.
git clone url regenera un repositorio de GIT a partir de uno de GITHUB y los vincula.
Fork crea un repositorio a partir de uno de terceros en GITHUB.
Open pull request genera una solicitud para que el dueño del repositorio, del cual hicimos un fork, nos permita hacer un merge.
